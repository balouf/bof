Factor Tree¶

The factortree module contains the core engine of the Bag of Factors package.

class bof.factortree.FactorTree(corpus=None, auto_update=False, preprocessor=None, n_range=5, filename=None, path='.')[source]¶

Maintain a tree of factor of a given corpus.

Parameters

  • corpus (list of str, optional) – Corpus of documents to decompose into factors.

  • auto_update (bool, optional) – When processing external texts, should the tree be updated (TO BE MOVED WITH THE FACTORS MODULE WHEN CREATED).

  • preprocessor (callable) – Preprocessing function to apply to texts before adding them to the factor tree.

  • n_range (int or None, optional) – Maximum factor size. If None, all factors will be extracted.

count¶

Keep for each factor a dict that tells for each document (represented by its index) the number of occurences of the factor in the document.

Type

list of dict

edges¶

Keep for each factor a dict that associates to each letter the corresponding factor index in the tree (if any).

Type

list of dict

corpus_list¶

The corpus list.

Type

list of srt

corpus_dict¶

Reverse index of the corpus (corpus_dict[corpus_list[i]] == i).

Type

dict of str -> int

factor_list¶

The factor list.

Type

list of srt

factor_dict¶

Reverse index of the factors (factor_dict[factor_list[i]] == i).

Type

dict of str -> int

Examples

>>> corpus = ["riri", "fifi", "rififi"]
>>> tree = FactorTree(corpus=corpus)
>>> tree.self_factors
[8, 8, 15]

Save instance to file.

Parameters
